# Отображение поставщиков на странице товара

## Что сделано

### 1. Создан публичный API endpoint
**Файл:** `src/app/api/public/products/[id]/suppliers/route.ts`

- `GET /api/public/products/[id]/suppliers`
- Возвращает поставщиков с ценами, сроками доставки и наличием
- Только товары в наличии (`inStock: true`)
- Сортировка: сначала в наличии, потом по цене возрастание

### 2. Обновлена страница товара
**Файл:** `src/app/product/[slug]/product-client.tsx`

- Использует публичный endpoint вместо admin
- Загружает поставщиков при монтировании компонента
- Отображает через компонент `SupplierSelector`

### 3. Разрешён публичный доступ
**Файл:** `src/app/api/admin/products/[id]/suppliers/route.ts`

- Изменён комментарий (не защищено middleware)

## Как отображаются поставщики

### Блок "Ещё поставщики"
Отображается после блока цены и складов на странице товара.

**Для каждого поставщика показывается:**
- 🏷️ **Название** (WB, Ozon и т.д.) с иконкой
- 💰 **Цена** (крупным шрифтом)
- ~~Старая цена~~ (если есть скидка)
- 🏷️ **Процент скидки** (бейдж)
- ✅ **Наличие** ("В наличии" / "Нет в наличии")
- 🚚 **Срок доставки** (например "2–4 дн.")
- ⭐ **Рейтинг** и количество отзывов
- 🛒 **Кнопки:** "В корзину" и "Купить"
- 📋 **Раскрывающаяся информация** (по клику на стрелку)

### Пример отображаемых данных
```
Ещё поставщики               [3 поставщика]

🔵 Wildberries                              ⭐⭐⭐⭐☆ (150)
1693 ₽  ~~4990 ₽~~  [-66%]        ✅ В наличии
                                           🚚 2–4 дн.
[В корзину]  [Купить]  [▼]

🔵 Ozon
1750 ₽                             ✅ В наличии
                                   🚚 3–5 дн.
[В корзину]  [Купить]  [▼]
```

## Проверка работы

### 1. Проверьте что поставщики сохранены
```sql
SELECT * FROM "ProductSupplier" 
WHERE "productId" = 'cmnispa25001kqo9vt3xtsmvm'
ORDER BY price ASC;
```

### 2. Откройте страницу товара
`http://localhost:3000/product/asus-tuf-rtx-4070-ti-super-oc-16gb`

### 3. Проверьте консоль браузера
Не должно быть ошибок при загрузке `/api/public/products/.../suppliers`

### 4. Проверьте Network tab
- Запрос: `GET /api/public/products/{id}/suppliers`
- Ответ: `{ ok: true, data: [...] }`

## Отладка

### Поставщики не загружаются
1. Проверьте Network tab → статус ответа
2. Проверьте что поставщики есть в базе (SQL выше)
3. Проверьте консоль сервера на ошибки

### Поставщики пустые после парсинга
1. Убедитесь что товар сохранён с источниками
2. Запустите парсинг заново
3. Проверьте что webhook endpoint отработал успешно

## Архитектура

```
Парсинг (extension)
    ↓
Webhook `/api/admin/parser/webhook`
    ↓
saveSuppliers() → ProductSupplier
    ↓
Страница товара загружает
    ↓
GET `/api/public/products/[id]/suppliers`
    ↓
SupplierSelector компонент
    ↓
Отображение: цена, срок, наличие
```
