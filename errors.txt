## 📊 Ожидаемые ошибки при первом запуске

После установки зависимостей и первого запуска могут возникнуть следующие ошибки:

### 1. Отсутствие зависимостей

```bash
pnpm install
```

### 2. Не сгенерирован Prisma Client

```bash
pnpm db:generate
```

### 3. Не создана база данных

```bash
# Создание БД
docker exec -it pg-local psql -U postgres -c "CREATE DATABASE \"site-1000fps\";"

# Применение миграций
pnpm db:migrate
```

### 4. Отсутствуют .env файлы

```bash
cp .env.example .env
cp packages/api/.env.example packages/api/.env
cp apps/storefront/.env.local.example apps/storefront/.env.local
cp apps/admin/.env.local.example apps/admin/.env.local
```
