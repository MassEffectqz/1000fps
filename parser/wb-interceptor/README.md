# ⚡ 1000fps WB Парсер

Chrome расширение для парсинга и отслеживания цен на Wildberries с синхронизацией на сервер.

## 🚀 Возможности

- ✅ Добавление товаров по артикулу или ссылке
- ✅ Автоматическое обновление цен каждые 2 часа
- ✅ Уведомления об изменении цены (браузер)
- ✅ История изменения цен с графиками
- ✅ Полная страница трекера с аналитикой
- ✅ Компактный popup для быстрого доступа
- ✅ Экспорт данных в JSON
- ✅ Синхронизация с сервером 1000fps

## 📦 Установка

1. Откройте Chrome → `chrome://extensions/`
2. Включите **"Режим разработчика"** (переключатель в правом верхнем углу)
3. Нажмите **"Загрузить распакованное"**
4. Выберите папку `wb-interceptor`

## 🎯 Как использовать

### Быстрое добавление товара

1. Нажмите на иконку расширения в тулбаре Chrome
2. Введите артикул товара или вставьте ссылку на товар WB
3. Нажмите **+** или Enter
4. Товар добавлен в список

### Просмотр и управление

- **В popup** — быстрый список товаров с текущими ценами
- **Полная версия** — откройте страницу трекера с графиками и историей

### Обновление цен

- **Автоматически** — каждые 2 часа (фоновый режим)
- **Вручную** — нажмите кнопку "↻" в popup или трекере

### Экспорт данных

- Нажмите кнопку "📥" в popup или полной версии трекера
- Файл `wb-prices-ГГГГ-ММ-ДД.json` будет загружен
- Содержит все товары с историей цен и текущими данными

## 🔧 Как это работает

Расширение использует открытую вкладку `wildberries.ru` для получения данных о ценах через внутреннее API WB. Это обходит CORS-ограничения и не требует ручной авторизации.

Расширение автоматически синхронизируется с сервером 1000fps (по умолчанию `http://localhost:3001`):

- При загрузке расширения
- Каждые 30 секунд проверяет новые pending товары
- Отправляет обновлённые цены на сервер

- 📈 Цена выросла — красное уведомление

- **Чёрный** (#000000) — фон
- **Белый** (#ffffff) — текст
- **Оранжевый** (#ff6b00) — акценты, цены

- **Красный** (#ff4444) — повышение цены

## ❓ Решение проблем

### Товар не добавляется

- Убедитесь, что открыта вкладка `wildberries.ru`
- Проверьте правильность артикула
- Перезагрузите страницу WB

### Цены не обновляются

- Откройте вкладку WB (расширение работает через страницу)
- Нажмите "Обновить все" вручную
- Проверьте консоль Service Worker на ошибки (`chrome://extensions` → Service Worker)

### Extension не синхронизируется с сервером

- Проверьте, что сервер запущен на `http://localhost:3001`
- Проверьте консоль Service Worker на ошибки
