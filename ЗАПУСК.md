# 🚀 Запуск проекта 1000FPS

## ⚡ Быстрый старт

```bash
# 1. Перейти в директорию проекта
cd "C:\Users\danya\Documents\My Web Sites\1000fps - дубль 2"

# 2. Установить зависимости (если ещё не установлены)
pnpm install

# 3. Сгенерировать Prisma Client
cd packages/api
pnpm prisma generate
cd ../..

# 4. Запустить Docker (PostgreSQL + Redis)
pnpm docker:up

# 5. Запустить миграции БД
cd packages/api
pnpm prisma migrate deploy
cd ../..

# 6. Запустить ВСЕ сервисы
pnpm dev
```

---

## 🌐 После запуска

Откройте в браузере:

| Сервис | URL |
|--------|-----|
| **Магазин** | http://localhost:3000 |
| **Админка** | http://localhost:3002 |
| **API** | http://localhost:3001/api/v1 |
| **Swagger** | http://localhost:3001/swagger |
| **Prisma Studio** | http://localhost:5555 |

## 📦 Команды для разработки

### Запуск всех сервисов
```bash
pnpm dev
```

### Запуск по отдельности
```bash
pnpm dev:api      # Backend (порт 3001)
pnpm dev:store    # Storefront (порт 3000)
pnpm dev:admin    # Admin (порт 3002)
```

### Остановка
```bash
pnpm docker:down  # Остановить Docker контейнеры
# Ctrl+C для остановки dev серверов
```

## ⚠️ Решение проблем

### Ошибка: Port already in use
```bash
# Освободить порты
docker-compose down
pnpm docker:up
```

### Ошибка: Prisma Client not found
```bash
cd packages/api
pnpm prisma generate
```

### Ошибка: Database connection failed
```bash
# Проверить что Docker запущен
docker-compose ps

# Перезапустить контейнеры
pnpm docker:restart
```
