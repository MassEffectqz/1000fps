# 1000FPS WB Parser — Chrome Extension

Расширение для Chrome для парсинга цен Wildberries и отслеживания их изменений.

## Возможности

- **Автоматический парсинг** — при открытии страницы товара на Wildberries цена парсится автоматически
- **Отслеживание цен** — хранение истории изменений цен для каждого товара
- **Трекер цен** — веб-интерфейс для просмотра всех отслеживаемых товаров
- **Уведомления** — браузерные уведомления при изменении цен
- **Синхронизация с сервером** — отправка данных на backend 1000FPS
- **Multi-source парсинг** — поддержка резервных артикулов для fallback
- **Экспорт данных** — выгрузка в JSON

## Установка

### 1. Загрузить в Chrome

1. Откройте `chrome://extensions/` в браузере
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку `extension/` из проекта
5. Расширение появится в списке

### 2. Добавить иконки

Поместите файлы иконок в папку `extension/icons/`:

- `icon16.png` — 16x16 пикселей
- `icon48.png` — 48x48 пикселей
- `icon128.png` — 128x128 пикселей

Без иконок расширение будет работать, но уведомления будут без значка.

### 3. Настроить backend

По умолчанию расширение отправляет данные на `http://localhost:3002`.

Для изменения URL:
1. Откройте popup расширения
2. Нажмите **Настройки**
3. Укажите нужный URL backend'а

## Использование

### Popup расширение

Кликните на иконку расширения в панели браузера:

- **Автопарсинг** — вкл/выкл автоматический парсинг при открытии товаров WB
- **Автозакрытие вкладок** — закрывать вкладку после успешного парсинга
- **Статистика** — количество товаров, успешных проверок, ошибок, изменений цен
- **Обновить все цены** — запустить проверку всех отслеживаемых товаров
- **Открыть трекер цен** — открыть полную страницу с таблицей товаров

### Трекер цен

Полнофункциональная страница с:

- **Таблица товаров** — артикулы, названия, цены, остатки, сроки доставки
- **Поиск** — по артикулу или названию
- **Фильтры** — по статусу (в наличии, нет в наличии, цена изменилась)
- **Сортировка** — по дате, цене, названию, изменению цены
- **История цен** — график и таблица изменений для каждого товара
- **Экспорт** — выгрузка всех данных в JSON

### Автоматический парсинг

1. Убедитесь что **Автопарсинг** включён в popup
2. Откройте страницу товара на Wildberries (например, `https://www.wildberries.ru/catalog/12345678/detail.aspx`)
3. Дождитесь уведомления об успешном парсинге
4. Товар автоматически добавится в трекер

## Permissions

- `storage` — хранение данных о товарах и настройках
- `tabs` — определение открытых вкладок WB
- `notifications` — уведомления об изменении цен
- `alarms` — периодическая проверка цен
- `scripting` — выполнение скриптов на страницах WB

## Хранимые данные

Расширение сохраняет в `chrome.storage.local`:

- `wbPrices` — объект с товарами и историей цен
- `autoParseEnabled` — флаг автопарсинга (boolean)
- `autoCloseEnabled` — флаг автозакрытия вкладок (boolean)
- `backendUrl` — URL backend'а (string)

## Troubleshooting

### Товар не парсится

1. Убедитесь что вкладка Wildberries полностью загрузилась
2. Проверьте что `www.wildberries.ru` открыт в браузере
3. Посмотрите консоль Service Worker (`chrome://extensions` → Inspection views)

### Цены не обновляются

1. Откройте popup и нажмите **Обновить все цены**
2. Проверьте что backend запущен на `localhost:3002`
3. Проверьте логи в консоли Service Worker

### Нет уведомлений

1. Проверьте разрешения на уведомления в `chrome://settings/content/notifications`
2. Убедитесь что иконка `icons/icon128.png` существует
