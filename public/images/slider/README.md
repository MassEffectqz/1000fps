# Изображения для слайдера

Поместите в эту папку изображения для слайдов:

- `rtx-4090.png` — изображение видеокарты NVIDIA GeForce RTX 4090
- `ryzen-9950x.png` — изображение процессора AMD Ryzen 9 9950X
- `monitor-360hz.png` — изображение игрового монитора 360 Hz

**Рекомендуемые требования:**
- Формат: PNG с прозрачным фоном
- Размер: ~400x400 px
- Объект должен быть хорошо виден на тёмном фоне
